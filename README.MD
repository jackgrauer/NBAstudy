# NBASTUDY

## Project Overview
This analysis determines whether NBA revenue gains spill over to municipal tax revenues in team-hosting cities.

## Requirements
This code needs the following packages installed in order to run:
    pandas
    numpy
    matplotlib
    seaborn
    statsmodels
    scikit-learn
    scipy
    linearmodels
    patsy
    pingouin
